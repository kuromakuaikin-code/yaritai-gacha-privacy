# Fable 5 性能調査 10 — ゲームエフェクト

「Fable 5 × GPT image 2 × Blender でゲームエフェクトを作った」という報告の再現実験。
この実行環境には Blender MCP も GPT image 2 API も無いため、**同じ工程をすべてコードだけで置き換えて**作成した。

| 元の工程 | この環境での代替 |
| --- | --- |
| Blender MCP で3Dモデル作成 | JavaScript でジオメトリを直接生成（板ポリ・円弧メッシュ・ビルボード） |
| GPT image 2 でテクスチャ生成 | Canvas 2D による手続き生成（魔法陣・炎・光柱・衝撃波など9枚） |
| プレビュー画面 | 素の WebGL（ライブラリ不使用）＋ HTML/CSS |

## 見かた

`index.html` をブラウザで開くだけ（ビルド不要・依存ゼロ・1ファイル完結）。
スマホ対応：ドラッグで回転、ピンチでズーム。

GitHub Pages 公開後の URL:
`https://kuromakuaikin-code.github.io/yaritai-gacha-privacy/fable5-effects/`

## 収録シーン（8種）

- **炎** — ゆらめく火柱・火の粉・煙・地面の照り返し
- **爆発** — 火球・重力つき火花・拡がる衝撃波リング・画面フラッシュ
- **魔法陣** — 手続き生成した召喚陣テクスチャが2枚重ねで逆回転、光柱と螺旋の粒子
- **斬撃** — 円弧メッシュによる剣閃（2パス描画）と先端の火花
- **回復** — 螺旋を描いて昇る癒しの光と十字のきらめき
- **雷** — 分岐する落雷の生成と着弾の火花
- **緋色魔法** — スカーレット・ウィッチ風の混沌魔法。宙に浮かびカメラに正対する
  スパイク付きの陣（中央＋両手元の3枚）、緋色の魔力の渦、赤い放電、足元の暗赤の霧
- **スカーレット** — キャラクターシートから起こした魔女の3Dモデル（猫耳帽子・金髪・
  緋色のスカート・マント）。球・円錐台・チューブをコードで組み合わせ、ハーフランバート＋
  緋色の照り返しでライティング。顔は Canvas で描いたデカール。混沌魔法を詠唱中

## 実装メモ

- パーティクルは `gl.POINTS` のポイントスプライト（位置・サイズ・色・回転を毎フレーム転送）
- 色はライフタイムに沿ったカラーストップ補間、加算合成と通常合成を使い分け
- テクスチャは全て起動時に Canvas 2D で描いて `texImage2D` へ（プリマルチプライドα）
- カメラは自動オービット＋タッチ/マウス操作、`prefers-reduced-motion` 時は停止状態で開始
