# drafts — 掲載前の下書き置き場

このフォルダの内容は**サイトには表示されません**（events.json に転記して初めて公開されます）。

- `research-aichi-2026.json` などは、公式サイト・自治体サイト等の一次情報をもとに運営が収集した「掲載候補メモ」です。
- 掲載する場合は ADMIN.md の手順に従い、**必ず sourceUrl の公式情報を自分の目で再確認**してから、`events.json` に自分の言葉で書き直して追加してください。
- confidence が medium / low のものは日程未確定・未確認です。そのまま転記しないでください。
- 他のイベント情報サイトの紹介文・画像は転載禁止です（候補メモに含まれる desc も、掲載時は公式発表に基づき書き直すこと）。
