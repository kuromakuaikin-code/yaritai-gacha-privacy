# 面白いアプリ案 5

作成日: 2026-07-16

これまでの実績（やりたいガチャ／婚活デートメモ／盆踊りナビ愛知）から見えている勝ちパターンを前提に、次の一手を5案出し、最有力の1案はそのまま実装に入れるレベルまで具体化した。

## 前提：この開発体制の勝ちパターン

- **端末内保存・サーバー不要・プライバシー訴求**（審査も運用もラク、ポリシーも流用可）
- **買い切り¥160〜200のプレミアム＋AdMobバナー**（サブスクなしを売りにする）
- **Expo + EAS で Mac なしで App Store 提出**（app-expo の構成をそのまま流用できる）
- **日本のニッチな「人間関係×記録」領域**（婚活デートメモで実証済み）

## 候補5案

### 案A 🏠 親戚メモ 〜帰省・義実家の会話おたすけ帳〜（本命）

**一言で**: 婚活デートメモの「お相手」を「親戚・義実家」に置き換えたアプリ。帰省前の5分で「叔父さんの近況・前回話したこと・地雷話題」を見返せる。

- **痛み**: 盆・正月の帰省で「甥っ子の名前が出てこない」「義父の趣味を忘れた」「前も同じ話を聞いてしまった」は誰にでもある。特に配偶者側の親戚は覚えられない。
- **機能**: 親戚ごとのプロフィール（続柄・誕生日・好き嫌い・NG話題）／会った記録（いつ・何を話した・もらい物あげた物）／帰省前カンペ／「帰省の話題リスト」（婚活デートメモの話題エンジンを親戚向けに書き換え）
- **プレミアム**: 登録人数無制限（無料5人）＋話題全開放＋広告なし。¥200買い切り。
- **開発コスト**: ★☆☆ 最小。`app-expo` のコード（Db・Partners・Topics・IAP・課金導線）を **7割そのまま流用**できる。話題データの書き換えとラベル変更が中心。
- **時期**: いま7月。**8月のお盆帰省に間に合う**。12月末の年末帰省で2回目の山が来る。検索需要が年2回確実に発生する。
- **面白さ**: 「義実家対策アプリ」という切り口自体に話題性があり、SNS・メディアで拾われやすい。プライバシー重視（端末内のみ保存）が「こういうメモこそ絶対に漏れてほしくない」という文脈で強く効く。

### 案B 💸 おつきあい帳 〜ご祝儀・香典の記録〜

- **一言で**: 冠婚葬祭で「誰にいくら包んだ／もらった」を記録し、お返し・次に包む額をすぐ引ける帳簿。
- **痛み**: 紙のおつきあい帳文化は根強いがアプリの決定版がない。金額相場（続柄・地域別）を調べ直す人が毎回大量にいる。
- **プレミアム**: 記録無制限＋相場早見表＋マナー集。
- **評価**: 需要は堅いが対象年齢層が高くアプリ課金への抵抗がある。案Aの次に良い。開発コストは★★☆（帳簿UIと集計が新規）。

### 案C 🎰 推し活予算ガチャ

- **一言で**: 推し活（グッズ・遠征・ソシャゲ課金）の支出を推し別に記録し、「今月あといくら使えるか」をガチャ演出で見せる家計簿。
- **評価**: 市場は大きく「やりたいガチャ」のガチャ演出資産が活きるが、競合（推し活家計簿系）が既に複数あり、差別化が演出頼みになる。★★☆

### 案D 🏮 縁日・祭りスタンプ帳

- **一言で**: 盆踊りナビの個人版。行った祭り・盆踊りを写真1枚とスタンプで記録するコレクションアプリ。
- **評価**: 盆踊りナビとの相乗効果（サイト→アプリ導線）はあるが、季節が夏に偏り、課金理由が弱い。サイト側の機能として持つ方が良いかもしれない。★★☆

### 案E 💊 通院メモ 〜診察で聞きたいことカンペ〜

- **一言で**: 「診察室に入ると聞きたいことを忘れる」問題に特化。症状メモ・質問リスト・医師の回答記録。家族の分も管理。
- **評価**: 痛みは本物で端末内保存の訴求も強いが、健康系はストア審査・表現に気を使う。★★☆

## 推奨：案A「親戚メモ」を作る

理由は3つ。

1. **開発コストが圧倒的に低い**。婚活デートメモ（app-expo）の資産をほぼそのまま使い、実装の大半は「話題データの執筆」と「文言・配色の差し替え」で済む。
2. **季節の追い風が明確**。8月お盆・12月年末の年2回、確実に検索される。今から作れば初回の山に間に合う。
3. **既存アプリとカニバらない**。婚活デートメモの卒業生（結婚した人）がそのまま次の顧客になる。ストアの「デベロッパーの他のアプリ」導線が効く。

### MVP仕様（app-expo からの変更点）

| app-expo（婚活デートメモ） | 親戚メモ |
|---|---|
| お相手（Partners） | 親戚（続柄・所属の家・誕生日・好き嫌い・NG話題） |
| デート記録 | 会った記録（帰省・法事・電話。もらい物/あげた物欄を追加） |
| 話題リスト 5カテゴリ | 帰省話題 4カテゴリ：🍵久しぶりの挨拶から／🏠近況の聞き方／👶子ども・孫世代と／⚠️かわし方（結婚まだ？等への返し） |
| デート前カンペ | 帰省前カンペ（次の帰省日カウントダウン） |
| 無料3人まで | 無料5人まで（親戚は人数が多いので少し緩める） |
| premium ¥200 | premium ¥200（話題全開放・人数無制限・広告なし） |

- 保存キーは新規（例 `shinseki_memo_v1`）。データ構造は Db をコピーして partner→relative に読み替え。
- バンドルID例: `com.kuromakuaikin.shinsekimemo`
- プライバシーポリシーは date-memo のものを文言差し替えで流用（端末内保存・AdMobのみ）。
- 「⚠️かわし方」カテゴリ（「結婚は？」「二人目は？」への角が立たない返し集）がこのアプリの名物・バズ要素になる。プレミアムの目玉に置く。

### 次のアクション

1. `app-expo` を `shinseki-memo/` にコピーして文言・型・保存キーを差し替え
2. 帰省話題データ（4カテゴリ×10前後、good/flat/ng付き）を執筆
3. アイコン・ストア掲載文（「帰省前の5分で、親戚づきあいがラクになる」）
4. 7月中に審査提出 → お盆前に公開
